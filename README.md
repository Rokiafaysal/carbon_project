# Carbon Footprint Tracker - Flutter & AI Integration

A smart Flutter application designed to help users calculate their environmental impact (Carbon Footprint) by analyzing daily consumption data using Artificial Intelligence.

##  Key Responsibilities (My Role)
As the **Flutter Developer** for this project, I executed the following:
- **Responsive UI/UX:** Built fully responsive interfaces to ensure a seamless user experience across all mobile screen sizes and devices.
- **AI API Integration:** Connected the application to AI Models via REST APIs to transmit consumption data and handle real-time prediction responses.
- **Smart Chatbot:** Developed a responsive chat interface that communicates with the backend to provide instant, AI-driven environmental advice.
- **Data Flow Management:** Organized the codebase to facilitate efficient API communication and optimized app performance.

##  Tech Stack
- **Frontend:** Flutter (Dart)
- **Networking:** Http for robust API communication.
- **State Management:** Native Flutter State Management (`setState`).
- **Integration:** RESTful APIs (AI Models & Chatbot services).

##  Technical Highlight
This project focused heavily on **Data Flow Architecture**. I implemented logic to send complex user inputs to the API and performed structured `Parsing` of the `JSON Response` received from the AI model. This allowed for the dynamic display of data insights and personalized user recommendations.
